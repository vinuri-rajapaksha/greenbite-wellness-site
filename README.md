# 🌿 GreenBite: Wellness & Healthy Living Portal
GreenBite is a fully responsive, interactive multi-page web application designed to promote balanced meals, physical activity, and mindfulness.

## 🚀 Key Features
1. Interactive Home & Navigation
* **Hero Section:** Dynamic health slogans that auto-rotate using JavaScript.
* **Responsive UI:** A custom-built animated hamburger menu for seamless mobile navigation.
* **Health Tip of the Day:** Logic-based content that updates daily based on the current date.
* **Newsletter Subscription:** Persistent storage using localStorage.

2. Smart Healthy Recipes
* **Dynamic Catalog:** Recipes displayed as cards populated from JSON data.
* **Filter & Search:** Real-time filtering by category and name.
* **Detailed Modals:** Pop-up views featuring ingredient lists, step-by-step instructions, and detailed nutrition tables.

3. Precision Calorie & Macro Calculator
* **BMR & TDEE Logic:** Sophisticated JavaScript calculator that accounts for age, gender, activity level, and goals.
* **Macronutrient Breakdown:** Automatic calculation of Carbs (50%), Protein (20%), and Fat (30%) ratios.
* **Visual Feedback:** Results displayed using animated counters and progress bars.

4. Fitness & Mindfulness Tools
* **Workout Generator:** Randomized exercise plans based on body part and equipment selection.
* **Exercise Timer:** Built-in countdown with sound and visual animations.
* **Mindfulness Suite:** A guided breathing tool, Pomodoro/meditation timer, and ambient sound toggles.
* **Progress Tracking:** Tracks completed sessions via localStorage.

## 🛠 Technical Stack
* **Structure:** Semantic HTML5
* **Styling:** Modern CSS3 (Flexbox, CSS Grid, Custom Animations, and Media Queries)
* **Logic:** Vanilla JavaScript (DOM Manipulation, Event Listeners, and JSON handling)
* **Storage:** Browser localStorage for data persistence.
* **PWA:** Progressive Web App features for offline capability.

## 📱 UX/UI Highlights
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
* **Modern Aesthetics:** Clean color palette, Google Fonts, and smooth scroll-based animations.
* **Form Validation:** Custom error handling and success messages for the Contact & Feedback page.
